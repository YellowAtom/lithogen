# <img src="res/icon.png" style="width: 26px; height: 26px;"> LithoGen 
A lithophane generator with 3D preview and in-depth configuration written in C++ with efficiency and customizability in mind.

## Dependencies
- [Embded 1.2.19](https://github.com/batterycenter/embed)
- [Glad 2.0.8](https://github.com/dav1dde/glad)
- [GLFW 3.4](https://github.com/glfw/glfw)
- [GLM 1.0.1](https://github.com/g-truc/glm)
- [ImGui 1.91.8](https://github.com/ocornut/imgui)
